An alternative to using the scope attribute is to use id and headers attributes to create associations between headers and cells. The way they are used is as follows:

  1. You add a unique id to each element.
  2. You add a headers attribute to each element. Each headers attribute has to contain a list of the ids of all the elements that act as a header for that cell, separated by spaces.
